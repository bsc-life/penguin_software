PENGUIN dashboard tutorial

How to use the PENGUIN dashboard:

PENGUIN (Promoter-ENhancer-GUided Interaction Networks) is a graph-based approach to study protein interactions in chromatin contact regions. The approach is based on the identification of Enhancers-Promoter Protein-protein Interaction Networks (EPINs) that are protein interactions conditioned to chromatin contacts. PENGUIN detects one EPIN for each promoter engaged in enhancer contacts by integrating several sources of information:

  • H3K27ac-HiChiP promoter-enhancer contacts
  • RNA-seq gene expression
  • tissue-specific physical nuclear protein interactions
  • high-quality curated DNA binding motifs

The PENGUIN dashboard allows exploring the results of the application of the PENGUIN approach in prostate cancer (PrCa), where we identified and characterized a total of 4,314 promoter and the corresponding EPINs. In this tutorial we will journey through the main functionalities of this interactive web page.

Navigation bar

The navigation bar consists of three elements. The About popover shows general information about the PENGUIN approach. The Links downdrop menu shows external links to the Github repository of the dashboard and this documentation.

The Download the network button downloads a TSV (tab-separated) formatted file describing the network that is currently displayed.


Input field

The input field is placed on the left-hand side of the page and consists of a series of selectors. The Promoter selector is a dropdown menu that allows selecting the promoter for which the corresponding EPIN is displayed in the output field.


To better highlight the different elements of such network, additional selectors are available. The DBPs displayed size and the DBPs displayed color selectors are dropdown menus that allow changing size and color of the nodes corresponsing to DNA-binding proteins (DBPs), i.e. the circles that are displayed vertically on the left (indicated as "Promoter" side) and on the right (indicated as "Enhancer(s)" side) of the output image. The visualization methods that can be chosen are:

  • degree: visualization based on the number of edges that are incident to a node
  • expression_log2FoldChange: visualization based on log2FC values of the differential expression analysis of prostate tumor vs normal from GEPIA
  • expression_padj: visualization based on adjusted p-values of the differential expression analysis of prostate tumor vs normal from GEPIA
  • fisher_test_OR: visualization based on Odd Ratios of a Fisher's exact test
  • fisher_test_pval: visualization based on p-values of a Fisher's exact test
  • Node_druggability: visualization based on node druggability information extracted from IID

    • The Intermediates displayed size and the Intermediates displayed color selectors are dropdown menus that allow changing size and color of the nodes corresponsing to intermediate proteins, i.e. the circles that are displayed in the center of the output image between the "Promoter" side and the "Enhancer" side. The additional visualization methods that can be chosen are:

  • Betweenness_global: visualization based on global betweenness centrality
  • Degree_global: visualization based on global degree
  • Betweenness_in_cluster: visualization based on in-cluster betweenness centrality
  • Betweenness_out_cluster: visualization based on out-cluster betweenness centrality
  • Betweenness_enrichment: visualization based on global betweenness centrality enrichment
  • Degree_in_cluster: visualization based on in-cluster degree
  • Degree_out_cluster: visualization based on out-cluster degree
  • Degree_enrichment: visualization based on global degree enrichment
  • Betweenness_enrichment_pval: visualization based on significance of global betweenness enrichment (Fisher's exact test)
  • Degree_enrichment_pval: visualization based on significance of global degree enrichment (Fisher's exact test)

    • Two sliders, called Differential expression cutoff (GEPIA) [log2FC] and Expression cutoff (LNCaP) [FPKM], enable to drop nodes based on gene expression cutoffs that use log2FC values of the differential expression analysis of prostate tumor vs normal from GEPIA and FPKM values of LNCaP, respectively

    Finally, four toggle switches allow adding (right switched) or removing (left switched) specific types of nodes and edges. In particular, SNP paths (PrCa SNPs in enchancer binding motifs) controls the visualization of the set of edges that bridge promoter-bound DBPs and intermediate proteins with enhancer-bound DBPs with PrCa SNPs falling in their binding motif. SNP paths (PrCa SNPs in intermediate proteins) controls the visualization of the set of edges that bridge promoter-bound DBPs with enhancer-bound DBPs through intermediate proteins with PrCa SNPs falling in the genomic region of the corresponding coding gene.

    The remaining two toggle switches, namely Nodes enriched in cluster GWAS+ and Edges enriched in cluster GWAS+, control the visualization of nodes and edges that have been found to be enriched in the cluster GWAS+ (see manuscript for more details).

    Output field

    The output field is placed on the center of the page and consists of a picture of the network displayed based on the parameters of the input field. The picture can be downloaded by right clicking on it and select "Save Image As...". The color of the edges indicate network paths with PrCa SNPs in enhancer binding motif (orange), network paths with PrCa SNPs in the gene coding for the intermediate proteins (green), or both (purple). PrCa SNPs in enchancer binding motifs are indicated beside the name of the enhancer-bound DBP, while PrCa SNPs in intermediate proteins are indicated with an asterisk. The protein found to be enriched in the cluster GWAS+ are highlighted in bold. Finally, druggable proteins from DrugBank are indicated as triangles.
